Comisión 39465

Se trata de un simulador básico de un restaurante para pedido online. La idea es aplicar los conocimientos aprendidos durante el curso y demostrar su comprensión.

Se incluyen uso de:
- Objetos y Arrays. Métodos de Arrays.
- Funciones y condicionales.
- Generación del DOM de forma dinámica. Eventos (Básicamente .onclick que es el más relevante para este caso).
- Sintaxis avanzada.
- Uso de librerías Sweetalert2 y Toastify.
- Manejo de promesas con fetch. 
- Carga de datos desde un JSON local. Usé un JSON local ya que no encontré una buena API gratuita que genere un menú de restaurante, la que encontré tenía el servidor caido, se ve que era antigua. Por un momento que estaba teniendo problemas con el "CORS" casi desarrollo Adds desde Fake Store Api simplemente para incluir el uso de una API pero gracias a la ayuda de los chicos de CoderAsk descubrí que tenía que ver mi sitio abriéndolo desde live server, en lugar de simplemente abrirlo con doble click desde la carpeta.
